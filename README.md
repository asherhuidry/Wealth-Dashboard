# Investment Portfolio Tracker

A lightweight web app to build a portfolio across different asset classes (including equities) and instantly view:

- **Portfolio performance** (invested value vs. current value, gain/loss, return %)
- **Asset allocation** by class
- **Holding-level breakdown** with per-asset returns

## Run locally

Because this is a static app, you can open `index.html` directly or run a simple local server:

```bash
python -m http.server 4173
```

Then open <http://localhost:4173>.
